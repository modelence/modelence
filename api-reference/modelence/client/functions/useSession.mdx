> **useSession**(): `object`

Defined in: [packages/modelence/src/client/session.ts:114](https://github.com/modelence/modelence/blob/4b5d4e42580d348e4979d6d48843d3b7edb2c8bf/packages/modelence/src/client/session.ts#L114)

`useSession` is a hook that returns the current user, and in the future will also return other details about the current session.

## Example

```ts
import { useSession } from 'modelence/client';

function MyComponent() {
  const { user } = useSession();
  return <div>{user?.handle}</div>;
}
```
