# Installation

Learn how to install and set up Modelence in your project.

## Prerequisites

- Node.js 18 or higher
- npm or yarn

## Install via npm

```bash
npm install modelence
```

## Install via yarn

```bash
yarn add modelence
```

## Verify Installation

```typescript
import { Modelence } from 'modelence'

console.log('Modelence installed successfully!')
```
