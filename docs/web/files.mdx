Modelence provides built-in file storage through Modelence Cloud. You can upload, retrieve, list, and delete files directly from your server-side code.

## Uploading a File

Use [`uploadFile`](/api-reference/modelence/server/functions/uploadFile) to upload a file to Modelence Cloud:

```ts
import { uploadFile } from 'modelence/server';
import fs from 'fs/promises';

const data = await fs.readFile('./photo.png');

const file = await uploadFile({
  fileName: 'photo.png',
  contentType: 'image/png',
  data,
});

console.log(file.url); // Public URL to access the file
```

The function accepts an `ArrayBuffer` or `Uint8Array` as `data`, so you can also pass a buffer received from an HTTP upload:

```ts
import { uploadFile } from 'modelence/server';

// Inside a route handler
async function handleUpload({ req }, { user }) {
  const file = await uploadFile({
    fileName: req.file.originalname,
    contentType: req.file.mimetype,
    data: req.file.buffer,
  });

  return { data: { fileId: file.fileId, url: file.url } };
}
```

## Retrieving a File

Use [`getFile`](/api-reference/modelence/server/functions/getFile) to fetch metadata for a single file by its ID:

```ts
import { getFile } from 'modelence/server';

const file = await getFile('file_abc123');
console.log(file.fileName);   // 'photo.png'
console.log(file.contentType); // 'image/png'
console.log(file.size);       // size in bytes
console.log(file.url);        // public URL
console.log(file.createdAt);  // ISO timestamp
```

## Listing Files

Use [`listFiles`](/api-reference/modelence/server/functions/listFiles) to retrieve a paginated list of all files:

```ts
import { listFiles } from 'modelence/server';

const { files, total } = await listFiles({ limit: 20, offset: 0 });

console.log(`Showing ${files.length} of ${total} files`);

for (const file of files) {
  console.log(file.fileName, file.url);
}
```

Both `limit` and `offset` are optional and default to `50` and `0` respectively.

## Deleting a File

Use [`deleteFile`](/api-reference/modelence/server/functions/deleteFile) to permanently remove a file:

```ts
import { deleteFile } from 'modelence/server';

await deleteFile('file_abc123');
```

## FileRecord type

All file functions return a `FileRecord` object with the following fields:

| Field | Type | Description |
| ----- | ---- | ----------- |
| `fileId` | `string` | Unique identifier for the file |
| `fileName` | `string` | Original file name |
| `contentType` | `string` | MIME type (e.g. `"image/png"`) |
| `size` | `number` | File size in bytes |
| `url` | `string` | Public URL to access the file |
| `createdAt` | `string` | ISO 8601 timestamp of when the file was uploaded |
