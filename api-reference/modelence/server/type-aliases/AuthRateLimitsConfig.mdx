Per-action rate limit overrides for authentication endpoints.
Every field is optional — omitting it keeps the built-in default.

## Example

```typescript
startApp({
  auth: {
    rateLimits: {
      signup: { perIp15Minutes: 5, perIpPerDay: 50 },
    },
  },
});
```
