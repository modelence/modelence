## Setting up MongoDB and Voyage AI

Follow these steps to get started with Modelence and Voyage AI:

1. **Create an account** on [cloud.modelence.com](https://cloud.modelence.com/)

2. **Create an application and local environment** in the Modelence Cloud dashboard

3. **Open the Settings page** of your environment and click on **Setup Local Environment**

4. **Choose VoyageAI as template** from the available options

5. **Copy the command** from the "Create New Project" section

6. **Open the directory** where you want to create your project and run the command from the "Connect to Modelence Cloud" section

7. **Add your Voyage AI API key**:
   - Get your API key from [voyageai.com](https://www.voyageai.com/)
   - Add it as the value of `MODELENCE_VOYAGEAI_API_KEY` in your `.modelence.env` file

8. **Run the project** using the command:
   ```bash
   npm run dev
   ```
