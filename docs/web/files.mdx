Modelence provides built-in file storage through Modelence Cloud. Files are organized by visibility — either `public` (accessible via a permanent URL) or `private` (accessible only via a time-limited signed URL).

## Uploading a File

Use [`uploadFile`](/api-reference/modelence/server/functions/uploadFile) to request a presigned upload URL, then PUT the file directly to storage:

```ts
import { uploadFile } from 'modelence/server';
import fs from 'fs/promises';

const { uploadUrl, filePath } = await uploadFile({
  filePath: 'photo.png',
  contentType: 'image/png',
  visibility: 'public',
});

const data = await fs.readFile('./photo.png');
await fetch(uploadUrl, {
  method: 'PUT',
  body: data,
  headers: { 'Content-Type': 'image/png' },
});

console.log(filePath); // e.g. 'public/photo.png'
```

The `visibility` field controls access:
- `"public"` — the file is accessible via a permanent URL
- `"private"` — the file requires a signed URL to access (default)

## Getting a File URL

Use [`getFileUrl`](/api-reference/modelence/server/functions/getFileUrl) to get a URL for displaying or linking to a file. For public files this returns a permanent URL; for private files it returns a time-limited presigned URL:

```ts
import { getFileUrl } from 'modelence/server';

const { url } = await getFileUrl('public/photo.png');
console.log(url); // Permanent public URL

const { url: privateUrl } = await getFileUrl('private/report.pdf');
console.log(privateUrl); // Time-limited presigned URL
```

## Downloading a File

Use [`downloadFile`](/api-reference/modelence/server/functions/downloadFile) to get a presigned download URL for a file:

```ts
import { downloadFile } from 'modelence/server';

const { downloadUrl } = await downloadFile('private/report.pdf');
// Redirect the user to downloadUrl or fetch it server-side
```

## Deleting a File

Use [`deleteFile`](/api-reference/modelence/server/functions/deleteFile) to permanently remove a file:

```ts
import { deleteFile } from 'modelence/server';

await deleteFile('public/photo.png');
await deleteFile('private/report.pdf');
```

## File paths

File paths always include the visibility prefix: `public/<path>` or `private/<path>`. The `uploadFile` function returns the full `filePath` (including prefix) in its result, which you can store and pass to the other functions.

```ts
const { filePath } = await uploadFile({ filePath: 'avatars/user-123.png', contentType: 'image/png', visibility: 'private' });
// filePath === 'private/avatars/user-123.png'

const { url } = await getFileUrl(filePath);
```
