# Getting Started

Welcome to Modelence! This guide will help you get up and running quickly.

## Installation

Install Modelence via npm:

```bash
npm install modelence
```

## Quick Start

Here's a simple example to get you started:

```typescript
import { Modelence } from 'modelence'

const app = new Modelence({
  // Your configuration
})

app.start()
```
